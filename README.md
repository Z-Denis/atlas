# atlas

[![CI](https://github.com/Z-Denis/atlas/actions/workflows/ci.yml/badge.svg)](https://github.com/Z-Denis/atlas/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

A Rust framework for variational methods on structured configuration spaces.

Atlas provides composable abstractions for configuration spaces, local spaces, homogeneous product spaces, probabilistic models, wave functions, sampling algorithms, and variational optimisation. It is designed for high-performance scientific computing with a minimal, generic core.

> **Status:** Early development. The API is not yet stable.
